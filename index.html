TaskSort

TaskSort is a simple CLI task manager written in Python that parses commands with argparse and stores data in a JSON file. It allows you to create, complete, delete, and list tasks.

Features: Add and save tasks to JSON storage, delete tasks from storage, list tasks loaded from storage, and complete a task

Technologies: Python, JSON, argparse
